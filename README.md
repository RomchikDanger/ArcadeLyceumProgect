Мой проект "Футбол" представляет собой двухмерную игру, написанную на Python с использованием библиотеки Arcade. Я выбрал эту тему, потому что футбол — популярный вид спорта, а создание игры с позволяет наглядно продемонстрировать понимание важных концепций программирования. Игрок управляет синим кругом (футболистом), который может перемещаться по полю и бить по оранжевому мячу, стараясь забить гол в ворота соперника.
Отдельное внимание я уделил системе хранения результатов. Все итоги матчей сохраняются в базу данных SQLite: имя игрока, количество забитых и пропущенных голов, общий счет и дата матча. В игре есть три экрана: главное меню с кнопками, сама игра и таблица рекордов, где отображаются лучшие результаты.
В результате получилась полностью работоспособная игра, в которую можно играть, соревноваться с самим собой и улучшать свои результаты.
